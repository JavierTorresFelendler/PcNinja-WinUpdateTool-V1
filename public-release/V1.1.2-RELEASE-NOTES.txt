PcNinja WinUpdate Tool V1.1.2 Release Notes

Changed since V1.1.1:
- Fixed false Windows Update reset failure: "Argument types do not match".
- The reset could already stop services, delete/recreate C:\Windows\SoftwareDistribution, and restart services correctly, but then fail while building the final service-status result object.
- Reset service status now returns a normal PowerShell array, so JSON output and success detection work correctly.

Existing V1.1 behavior remains:
- Reset uses the Snooz-style service stop behavior first.
- Reset deletes and recreates the full C:\Windows\SoftwareDistribution cache folder.
- Reset restarts Windows Update related services and reports cache/service state.
- MSI installer with interactive UI and silent deployment support.
- Portable EXE with GUI and CLI support.
